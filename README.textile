h1. Picfair P5JS play

30 mins playing with image processing in the browser:

"https://picfair-p5js-play.firebaseapp.com":https://picfair-p5js-play.firebaseapp.com

h2. Running Locally

bc. $ git clone git@github.com:msp/picfair-p5js-play.git
$ cd picfair-p5js-play
$ python -m SimpleHTTPServer

Then open "http://localhost:8000":http://localhost:8000

h2. Deploy

Assuming you have a Node install:

bc. $ npm install -g firebase-tools
$ firebase deploy
